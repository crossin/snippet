**所用工具：**

Python3 

绘图：turtle 库（内置）

演奏+可视化：pgzero 库（需安装）

**代码说明：**

- draw.py - 绘图。Python环境无需安装库可直接运行
- play.py - 演奏音乐。需安装 pgzero 库
- draw_and_play.py - 绘图+演奏（不同电脑运行速率可能会有差异）
- visual_play.py - 演奏音乐 + 音符可视化 + 控制台输出
